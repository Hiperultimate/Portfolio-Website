Image Reel

Introduction

Image Reel is an image sharing website which aims to give users the ability to stay anonymous or have a designated username while uploading images in case they want to secure their privacy.

Reason behind Image Reel

I wanted to make an image sharing website where the users are free to upload things they like without getting any backlash from others users , creating fake accounts is an alternative, but I felt like giving everyone a fair playing ground is the way to go.

Project Details

Image Reel was made using Java, CSS and HTML.


Technologies used:

  • Java Database Connectivity (JDBC)
  • Java Server Pages (JSP)
  • Servlet for Local Hosting
  • MySQL to serve as a Database
  • Bootstrap

Features:

  • Database Connectivity
  • Account Registration
  • Image Storing
